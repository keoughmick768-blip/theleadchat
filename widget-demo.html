🤖 NineM's AI Chat Widget

Embeddable AI chatbot for any website

⚡ Quick Start

Add this to any HTML page:

<script src="https://your-server.com/chatbot-widget.js" 
  data-api="https://your-server.com"
  data-business="Your Business Name"></script>

🎨 Customization Options

Configure the widget with these data attributes:

data-business - Business name
"My Company"
data-theme - Light or dark theme
"dark" or "light"
data-color - Primary brand color
"#667eea"
data-position - Widget position
"bottom-right", "bottom-left", "top-right", "top-left"
data-greeting - Initial message
"Hi there! How can I help?"
data-model - Ollama model to use
"qwen2.5:7b"
data-auto-open - Open on page load
"true"

📱 Programmatic Control

Control the widget with JavaScript:

// Open/close widget
NineMChat.open();
NineMChat.close();

// Send a message programmatically
NineMChat.send("Hello!");

// Update configuration
NineMChat.setConfig({ businessName: "New Name" });
